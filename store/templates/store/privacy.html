Ochrana osobních údajů (Privacy Policy)

Tento web shromažďuje pouze nezbytné osobní údaje pro vytvoření uživatelského účtu a umožnění nákupu či prodeje vinylových desek. Respektujeme zásady GDPR a všechna data jsou používána výhradně pro fungování webu.

Jaké údaje ukládáme

  • Jméno a e-mailová adresa (při registraci)
  • Informace o vložených vinylových deskách
  • Technická data potřebná pro zabezpečení a provoz webu

Co s daty děláme

Údaje nesdílíme s žádnými třetími stranami. Server je používá pouze pro přihlašování, správu účtu a funkce webu.

Vaše práva

  • Požádat o výmaz účtu
  • Požádat o opravu údajů
  • Požádat o výpis uložených informací

Pokud máte dotazy, kontaktujte nás prostřednictvím e-mailu uvedeného v kontaktech.
